
# Boehm Risk Management


Boehm's Risk Management Model is a proactive approach to identifying, addressing, and mitigating risks in project management, particularly within the context of software development. This model, developed by Dr. Barry Boehm in the 1980s, is pivotal in the domain of software engineering for its comprehensive framework in managing risks.

This tool allows users to quickly generate reports based on Boehm's Risk Management Model with a graphical interface based on Vue and Vuetify.
## Tech Stack

**UI:** Vue, Vuetify

**Backend:** Node, Electron


## Deployment

To compile this project for production run

```bash
  npm run electron:build
```

You can also build it for development with Vue hot reload with the following command
```bash
  npm run electron:serve
```
